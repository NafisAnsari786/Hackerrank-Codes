**SOLUTION**

```sql
WITH HighestAVGSal AS (
    SELECT 
        DEPARTMENT, 
        SALARY,
        DENSE_RANK() OVER (PARTITION BY DEPARTMENT ORDER BY SALARY DESC) as dense_rank
    FROM EMPLOYEES
)
SELECT  
    DEPARTMENT,
    AVG(SALARY) AS AvgSal
FROM HighestAVGSal
WHERE dense_rank = 1
GROUP BY DEPARTMENT;
```
